# GemmaEdu Companion

![GemmaEdu Companion](https://via.placeholder.com/800x200/0A66C2/FFFFFF?text=GemmaEdu+Companion)

**Offline multimodal AI tutor** built with Gemma 4 for the Gemma 4 Good Hackathon (Future of Education track).

- Fully works offline on low-end phones/tablets
- Multimodal: photo of textbook → step-by-step explanation
- Voice-first interaction
- Privacy-first: nothing leaves the device

**Status:** MVP in progress (Day 1 prototype)
